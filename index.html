¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño web que se utiliza para dar estilo a páginas escritas en HTML o XML. También se le conoce como Hojas de Estilo en Cascada.


Importancia de JavaScript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

align-items Alinea los elementos en un contenedor flex. flex-start, flex-end, center, baseline, stretch align-items: center; Fuente
